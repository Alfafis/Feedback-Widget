## 📚 Sobre

O projeto NLW Return Impulse foi desenvolvido com o objetivo de testar minha habilidades a parte de um curso da RocketSeat.

---

## 🚀 Tecnologias Utilizadas

- Typescript
- TailWindCss
- Node
- Prisma
- PostgreSql
- Vite
- Expo
- ReactNative
- ReactJs

**ATENÇÃO❗** É necessário que já esteja instalado:

- Node.js
- Git
- Gerenciador de pacotes (NPM ou Yarn).

---

## 📦 Como instalar e iniciar o projeto

### Web (Front-end)

```bash

# Entrar na pasta web
$ cd web/

# Instalar as dependências
$ npm install

# Iniciar a aplicação
$ npm run dev

```

---

### Server (Back-end)

```bash

# Entrar na pasta server
$ cd server/

# Instalar as dependências
$ npm install

# Iniciar a aplicação
$ npm run dev

```

---

### 📋 Testes
```
# Entrar na pasta server
$ cd server/

# Iniciar o teste
$ npm run test
```

---

### Mobile (Mobile)

```bash

# Entrar na pasta mobile
$ cd mobile/

# Instalar as dependências
$ npm install

# Iniciar a aplicação
$ expo start

```
